# Comic-Con India — Parking Management System
## ER Diagram

## Overview
This project contains the Entity Relationship Diagram
for a multi-zone event parking system designed for
Comic-Con India.

## Tool Used
Eraser.io — Diagram as Code (ERD syntax)

## Tables (13 Total)
- vehicle
- vehicle_category
- parking_zone
- parking_spot
- spot_category
- spot_reservation
- reservation_type
- parking_ticket
- parking_session
- payment
- price_rate_vehicle
- event_day
- coupon_code

## Key Features Covered
- Multi-day event tracking (event_day)
- Reserved spots for VIP, Staff, Cosplayers
- EV charging spot support
- Coupon/discount system
- Entry and exit session tracking
- Payment with method and status
- Spot reuse across multiple sessions
- Vehicle type based pricing
